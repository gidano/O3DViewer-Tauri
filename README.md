# O3D Viewer – Windows / Tauri wrapper

Online3DViewer 0.18.0 packaged as a small native Windows application.

## Features

- Explorer drag-and-drop into the viewer window.
- File → Open.
- Windows file associations for STL, OBJ, 3MF, FBX, GLB/glTF, STEP/STP, IGES/IGS, IFC, FreeCAD and the other declared formats.
- Double-clicking an associated file passes that exact file to the viewer.

## Build

The GitHub Actions workflow creates an NSIS installer under **Artifacts**.
